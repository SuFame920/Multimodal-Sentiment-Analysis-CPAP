# Multimodal-Sentiment-Analysis-CPAP
Aiming at the problem that false correlations exist within modalities in multimodal sentiment analysis and that intermodal sentiment ambiguities can mislead false correlation mitigation, this paper proposes a multimodal sentiment analysis model based on the fusion of causal perception enhancement and ambiguity perception.
